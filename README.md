# vtt_converter

1. 日本語文字起こしスプレッドシートの「webvtt」シートの内容をtsvでダウンロード
    1. ファイル名の例："2240_1_2_1.xlsx - webvtt.tsv"
2. inputsフォルダにtsvファイルを格納
3. vtt_converter.pyを実行する
4. outpusフォルダにVTTファイルが出力される
    1. ファイル名の例："2240_1_2_1_ja.vtt"
